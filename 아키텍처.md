## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | SNAP (패션 소셜 미디어) |
| 타입 | React 웹 + Android 하이브리드 앱 (Capacitor) |

| 빌드 도구 | Vite 7.3.1 |
| 스타일링 | Tailwind CSS v4.2.1 |

| HTTP 클라이언트 | Axios v1.13.6 |

### main.jsx - 프로바이더 계층 구조
```
<StrictMode>                    ← React 개발 경고 + 이중 렌더링 감지
  <AuthProvider>                ← 인증 상태 (user, token) - 최외곽
    <AlertProvider>             ← 전역 모달/확인 다이얼로그
      <BrowserRouter>           ← URL 라우팅
        <App />                 ← 라우트 정의

  </AuthProvider>
```

**주의:** `AuthProvider`가 최외곽이어야 하는 이유 → AlertProvider 및 BrowserRouter 하위 컴포넌트 모두 `useAuth()` 사용 가능

```javascript
    document.documentElement.classList.remove('dark');
  }
}, []); // 마운트 시 1회만 실행
```

## 6. Context 프로바이더 (전역 상태)

### AuthContext (인증 상태)

```javascript
// 제공 값
{
  user: UserObject | null,           // 현재 로그인 사용자 (null=미로그인)
  isAuthenticated: boolean,          // !!user (파생 값)
  isLoading: boolean,                // localStorage 세션 복원 중 true
  login(token, userData) → void,     // localStorage + state 업데이트
  logout() → void,                   // token/user 삭제
  updateUser(data) → void,           // 사용자 정보 부분 업데이트
  checkAuth() → Promise<void>        // localStorage에서 세션 복원
}

// localStorage 구조
localStorage['authToken'] = "eyJhbGc..."       // JWT 문자열
localStorage['user']      = '{"userId":42,...}' // JSON 문자열
```

**세션 복원 흐름:**
1. AuthProvider 마운트 → `checkAuth()` 호출
2. localStorage에서 `authToken` + `user` 읽기
3. 둘 다 존재 시 JSON 파싱 후 state 설정
4. `isLoading = false` (성공/실패 모두)
5. ProtectedRoute: `isLoading=true`이면 스피너, `isAuthenticated=false`이면 `/login`으로 이동

### AlertContext (전역 모달)

```javascript
// 제공 값
{
  showAlert(message, title?, type?) → void,       // 단순 알림 모달 표시
  showConfirm(message, onConfirm, title?) → void, // 확인/취소 모달 (onConfirm 콜백)
  closeAlert() → void                             // 모달 닫기
}

// 모달 타입 & 아이콘
'alert'   → 빨간 아이콘 (에러/경고)
'success' → 초록 아이콘 (성공 확인)
'info'    → 파란 아이콘 (정보/확인 요청)

// UI 구조
- z-index 9999 전체 화면 오버레이
- 흐림(blur) 배경 (클릭 시 닫기 가능)
- 중앙 카드: 아이콘 + 제목 + 메시지 + [취소/확인] 버튼
```

## 7. API 서비스 계층 (src/api/)

### apiClient.js (Axios 인스턴스)

```javascript
// 환경 감지 및 baseURL 결정
getBaseUrl():
  localhost / 127.0.0.1 → '/api' (Vite 프록시 사용)
  Android 에뮬레이터     → 'http://10.0.2.2:8080/api' (특수 IP)
  기타                   → VITE_API_URL 환경변수 또는 'http://10.0.2.2:8080/api'

withCredentials: true   // 쿠키 포함 CORS
timeout: 10000          // 10초 타임아웃

// Request Interceptor
Authorization: Bearer {localStorage.getItem('authToken')} 자동 주입

// Response Interceptor - 401 처리
TOKEN_EXPIRED → 알림 표시 → localStorage 토큰 제거 → /login 리다이렉트
기타 오류 → caller로 전파
```

```javascript
export default defineConfig({

        changeOrigin: true,
```
